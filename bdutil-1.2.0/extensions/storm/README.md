Deploying Apache Storm on Google Compute Engine
===============================================

Basic Usage
-----------

Basic installation of [Apache Storm](https://storm.apache.org/) alongside Hadoop on Google Cloud Platform.

    ./bdutil -e extensions/storm/storm_env.sh deploy

Or alternatively, using shorthand syntax:

    ./bdutil -e storm deploy

Status
------

This plugin is currently considered experimental and not officially supported.
Contributions are welcome.
